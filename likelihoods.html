class revrand.likelihoods.Bernoulli¶

Bernoulli likelihood class for (binary) classification tasks.

A logistic transformation function is used to map the latent function from the GLM prior into a probability.

Ey(f)¶

Expected value of the Bernoulli likelihood.

Parameters:f (array_like) – latent function from the GLM prior (\(\mathbf{f} = \boldsymbol\Phi \mathbf{w}\))
Returns:Ey – expected value of y, \(\mathbb{E}[y|f]\).
Return type:array_like
bounds¶

Get this object’s parameter bounds. This is a list of pairs of upper and lower bounds, with the same length as the total number of scalars in all of the parameters combined (and in order).
