revrand.basis_functions.base.FastFood¶

class revrand.basis_functions.base.FastFood(nbases, Xdim, lenscale_bounds=Positive(lower=1e-14, upper=None))¶

Fast Food basis function, which is an approximation of the random radial basis function for a large number of bases.

This will make a linear regression model approximate a GP with an RBF covariance function.

__init__(nbases, Xdim, lenscale_bounds=Positive(lower=1e-14, upper=None))¶

Construct a random radial basis function (RBF) object.

Parameters:
  • nbases – a scalar for how many random bases to create approximately, this actually will be to the neareset larger two power.
  • Xdim – the dimension (d) of the observations. lenscale_bounds: a tuple of bounds for the RBFs’ length scales.
