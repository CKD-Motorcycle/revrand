revrand.utils.base.append_or_extend¶

revrand.utils.base.append_or_extend(mylist, *objs)¶

Append an object to a list, or extend the list if the object is a list.

Parameters:
  • mylist (list) – list to extend
  • objs (object(s)) – object(s) to append to the list if not a list, else concatenate the lists
Returns:

the extended list.
Return type:

list
