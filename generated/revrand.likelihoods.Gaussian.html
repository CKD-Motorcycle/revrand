revrand.likelihoods.Gaussian¶

class revrand.likelihoods.Gaussian(var_bounds=Positive(lower=1e-14, upper=None))¶

A univariate Gaussian likelihood for general regression tasks.

No transformation function is needed since this is conjugate to the GLM prior.
