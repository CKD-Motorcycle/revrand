revrand.likelihoods.Gaussian¶

class revrand.likelihoods.Gaussian(var_init=<revrand.btypes.Parameter object>)¶

A univariate Gaussian likelihood for general regression tasks.

No transformation function is needed since this is (conditionally) conjugate to the GLM prior.

Parameters:var_init (Parameter, optional) – A scalar Parameter describing the initial point and bounds for an optimiser to learn the variance parameter of this object.
