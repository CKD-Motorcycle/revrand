revrand.basis_functions.base.PolynomialBasis¶

class revrand.basis_functions.base.PolynomialBasis(order, include_bias=True)¶

Polynomial basis class, this essentially creates the concatenation, Phi = [X^0, X^1, ..., X^p] where p is specified in the constructor.

__init__(order, include_bias=True)¶

Construct a polynomial basis object.

Parameters:
  • order – the order of the polynomial to create, i.e. the last power to raise X to in the concatenation Phi = [X^0, X^1, ..., X^order].
  • include_bias – If True (default), include the bias column (column of ones which acts as the intercept term in a linear model)

Methods

__init__(order[, include_bias]) Construct a polynomial basis object.
from_vector(X, vec) Apply this basis function to X like __call__(), but instead of being given parameter arguments, this function is given a flat list of all of the parameters.
grad(X) Return the gradient of the basis function w.r.t.each of the parameters.
grad_from_vector(X, vec) Return the gradient of the basis function w.r.t.each of the parameters, but like from_vector, instead of being given parameter arguments, this function is given a flat list of all of the parameters.
