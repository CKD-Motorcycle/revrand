Installation
============

Requirements
------------

Minimal
*******

- numpy
- scipy

Optional
********

Nonlinear Optimization (mainly for `BOBYCA`)
  - nlopt

Downloading and cachin datasets
  - requests
  - unipath

Visualization and interactive computing
  - matplotlib
  - ipython[notebook]

Development
***********

Documentation and deployment
  - sphinx
  - ghp-import

Testing
  - pytest
  - tox
